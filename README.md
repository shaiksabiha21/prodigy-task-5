# Task 5: Traffic Accident Data Analysis

## Dataset Information
* **Dataset Name:** US Accidents (A Countrywide Traffic Accident Dataset)
* **Dataset Source:** Kaggle
* **Note on Dataset Availability:** Due to GitHub's 25MB file size limit for web uploads, the raw dataset file (`US_Accidents_March23.csv`) is not included directly in this repository. 

## Project Overview
This project analyzes countrywide traffic accident data to uncover critical patterns relating to road conditions, weather factors, and the time of day when accidents occur most frequently. 

## Generated Visualizations Included
* `accident_time_distribution.png` (Hourly patterns)
* `accident_weather_factors.png` (Weather contributing factors)
* `accident_state_hotspots.png` (Geographic accident hotspots)
